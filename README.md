# Sapore Italiano - Sistema de Registro de Empleados

Aplicación de gestión de empleados y registro de jornada laboral para **Sapore Italiano**, restaurante ubicado en Plaza Alameda, Coín (Málaga).

## 🍝 Características

- **Dashboard** - Panel de control con estadísticas en tiempo real
- **Fichaje** - Registro de entrada/salida/pausas conforme al RD-Ley 8/2019
- **Historial** - Consulta y exportación de registros de jornada
- **Empleados** - Gestión completa del personal
- **Nóminas** - Visualización y gestión de nóminas

## 🛠️ Tecnologías

- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Base de datos:** Baserow
- **Deploy:** PM2

## 📋 Cumplimiento Legal

Sistema conforme al **RD-Ley 8/2019**:
- Registro diario de jornada
- Control de pausas y descansos
- Conservación de datos 4 años
- Accesible para Inspección de Trabajo
